# 🥽 Google Glass — Projeto HTML/CSS

> ⚠️ Projeto em andamento

Site estático desenvolvido durante meus estudos de **HTML e CSS**, como parte do curso do **Curso em Vídeo** (professor Gustavo Guanabara). O objetivo é praticar estruturação semântica de páginas, uso de tabelas, formulários, navegação e estilização com CSS.

## 🎯 Objetivo do projeto

Recriar uma página informativa sobre o Google Glass, aplicando conceitos como:

- Estrutura semântica (`header`, `nav`, `section`, `article`, `aside`, `footer`)
- Tabelas com `rowspan`/`colspan`
- Links internos e externos
- Organização de menu de navegação
- Estilização com CSS externo

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (em aplicação)

## 📚 Créditos

Projeto desenvolvido como exercício prático do curso **HTML5 e CSS3** do [Curso em Vídeo](https://www.cursoemvideo.com/), ministrado por **Gustavo Guanabara**. O conteúdo e o design original da página fazem parte do material didático do curso; este repositório documenta minha prática e evolução durante o aprendizado.

## 🚧 Status

Em desenvolvimento — já iniciada a aplicação de **JavaScript** para adicionar interatividade à página. Próximos passos incluem finalizar a folha de estilo (`estilo.css`), adicionar as páginas internas (`specs.html`, `multimidia.html`, `fale-conosco.html`) e revisar a responsividade.
